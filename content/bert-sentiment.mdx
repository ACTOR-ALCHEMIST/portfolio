Sentiment analysis is a useful exercise because the model quality is only one part of the experience. The interface needs to make confidence, edge cases, and unexpected predictions easy to inspect.

The workflow I prefer is simple: prepare balanced examples, train a baseline, collect error cases, then expose the model through a small demo so iteration is fast.
